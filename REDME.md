# Computer Class Scheduler

Данный проект представляет собой прототип системы составления расписания для компьютерного класса.

## Функциональность
- Создание расписания на неделю
- Учет заявок пользователей (время, день, группа, преподаватель)
- Разрешение конфликтов по приоритету групп
- Модульное тестирование (NUnit)

## Технологии
- C#
- .NET
- NUnit
- Git, GitHub

## Запуск проекта
1. Открыть решение в Visual Studio
2. Запустить unit-тесты
3. Использовать классы логики для формирования расписания

## Тестирование
Все ключевые компоненты покрыты unit-тестами:
- Проверка конфликтов
- Проверка приоритетов
- Проверка алгоритма расписания
